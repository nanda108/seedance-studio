## Структура актов (для роликов дольше 30с)

АКТ I — ЗАВЯЗКА (~20%)
  Сцена 1: [Название] — [1 строка] — [длительность]
АКТ II — РАЗВИТИЕ (~60%)
  Сцена 2: ...
АКТ III — РАЗВЯЗКА (~20%)
  Сцена N: ...

Для одного клипа (<=15с) — сразу таблица шотов без актов.

## Таблица шотов

СЦЕНА [N]: [НАЗВАНИЕ]
Локация: [где] | Время: [когда] | Тон: [настроение]

| # | ТИП | КАДР | КАМЕРА | ДЕЙСТВИЕ | DUR |
|---|-----|------|--------|----------|-----|
| 1 | Establishing | EWS | slow crane up | [описание] | 5s |
| 2 | Insert | ECU | static | [деталь] | 3s |
| 3 | Character | MS | slow push-in | [персонаж] | 4s |

Типы: Establishing / Master / OTS / Insert / Reaction / POV / Cutaway / Transition
Размеры: EWS -> WS -> MS -> MCU -> CU -> ECU

## Камера

Если camera: auto — выбирай из knowledge/cinematography.md по жанру и тону сцены.
Если есть предпочтения — применяй, адаптируй под каждый шот.

## Правила ритма

- Не больше 4-6 событий на 15 секунд
- Двойной контраст на каждом cut: меняй И кадр И тип камеры
  OK: WS handheld -> ECU static -> MS crane -> CU tracking
  NO: WS handheld -> MS handheld -> CU handheld

## Флаги

[NEEDS KEYFRAME] — первый шот с новым персонажем / уникальной локацией
[BEAT-SYNC] — монтаж под ритм
[COMPLEX CAM] — нестандартное движение
